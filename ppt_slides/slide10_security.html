Security & Trust: Non-Custodial by Design

No Custody of Funds

We never hold user tokens. Voting power is queried from chain, not stored.

Off-Chain Signatures Only

EIP-712 structured signing. No smart contract interactions, no gas fees, no exploit risk.

Verified Data Sources

All proposal data from official Snapshot API. All voting power from on-chain RPC calls.

AI Safety Measures

  • "Advisory Only" disclaimers on all AI output
  • Community flagging for bad analysis
  • Human review for high-value proposals (>$10M)

API Security

  • Rate limiting: 100 req/min per IP
  • CORS whitelist
  • Zod input validation
  • No SQL injection risk (parameterized queries)
